## Verify the installation

Confirm the CLI is now installed and at a compatible version:

```sh
codev --version
```

The reported version must be **at least as new as this extension** — that's
the compatibility rule Codev enforces.

Then tell the extension to re-check:

- Run [Codev: Recheck CLI](command:codev.recheckCli) from here, **or**
- Click the refresh button on the **Codev CLI** row in the Codev sidebar's
  **Status** view, **or**
- Open the Command Palette and run **Codev: Recheck CLI**.

When the check passes, the Status row shows a green check with the detected
version, and Codev commands are ready to use.
