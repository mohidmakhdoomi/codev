## Install the Codev CLI

> **Prerequisite:** Node.js **20 or newer** must be on your `PATH`
> (VS Code's own bundled Node doesn't count). Get it from
> [nodejs.org](https://nodejs.org). Check with `node --version`.

Install the CLI globally with npm:

```sh
npm install -g @cluesmith/codev
```

This provides the `codev`, `afx`, `porch`, and `consult` commands on your
`PATH`. The same command **upgrades** an existing install to the latest
version.

> Tip: if the extension detected an outdated CLI, it offers an **Update via
> npm** button on the notification that runs this command for you in an
> integrated terminal.

For other install methods and troubleshooting, see the
[installation docs](https://github.com/cluesmith/codev#installation).

Once the command finishes, continue to **Verify the installation**.
